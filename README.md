# HangmanGame

Petit jeu du pendu en console et peut-être bientôt avec une interface.

## Utilisation

Get the game ⤵

	git clone https://github.com/Instelce/HangmanGame
	cd HangmanGame

	pip install -r requirements.txt

Run the game ⤵

	python main.py
